Performing operations based on scanned numbers

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
Create a Java program that scans in an integer number and the program will perform a certain arithmetical operation based on that number. If the input number is '1', the program will calculate the square of a second number you scan in. If the input number is '2', the program will calculate the cube of a second number you scan in. If the input number is '3', the program will divide 1 by the second number you scan in. For any other input number, the program simply scans in one more number but does nothing with it.
Sample Input:
1
3

Sample Output:
9.0


Sample Input:
2
2

Sample Output:
8.0



Memory limit: 256 MB
Time limit: 5 seconds
